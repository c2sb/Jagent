Monk is a translator for all things pray.
It also supports an embedded pray template format called CAOS2PRAY.
This encodes pray data inside .cos files, eliminating the extra .txt file.

These translations are possible:
  • Compiled PRAY -> Pray template (.txt)
  • Pray template -> Compiled PRAY
  • CAOS script with CAOS2PRAY -> Compiled PRAY
  • CAOS script with CAOS2PRAY -> Pray template (.txt)
